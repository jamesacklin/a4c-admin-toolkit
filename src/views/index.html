Overview

The Apps4Clubz Admin UI toolkit is a highly-modular design system for rapid web page development. It contains different materials that can be assembled into more complex page layouts.

This guide contains real working examples, code snippets, documentation, and style guidelines.

This UI toolkit is based on the Bootstrap framework. At the time of its authoring, Bootstrap was up to version 3.3.7. Since the toolkit "extends" Bootstrap rather than including it, a developer can manage Bootstrap versions independently of this tookitâ€”simply reference Bootstrap and the required libraries first, then include this toolkit. Compatibility with Bootstrap version 4 is untested.

Usage

Start by including the Bootstrap libraries and the necessary dependencies:

Bootstrap CSS (MaxCDN):

<link href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-BVYiiSIFeK1dGmJRAkycuHAHRg32OmUcww7on3RYdg4Va+PmSTsz/K68vbdEjh4u" crossorigin="anonymous">

jQuery (Google CDN):

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>

Bootstrap JS (MaxCDN):

<script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js" integrity="sha384-Tc5IQib027qvyjSMfHjOMaLkfuWVxZxUPnCJA7l2mCWNIpG9mGCD8wGNIcPD7Txa" crossorigin="anonymous"></script>

Then add the following files to your page:

<link rel="stylesheet" href="assets/toolkit/styles/toolkit.css">
<script src="assets/toolkit/scripts/toolkit.js"></script>
