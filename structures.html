Data Grids

Data Grid

Data grids are an alternative to table rows that group pieces of data together with a label. This allows for grouping a custer of data items together into the same cohesive area, and allowing the user to operate on the group of datum. This is particulary useful for nesting inside cards.

Data grids are built using the semantically awesome description list element (<dl>) and its corresponding description term (<dt>) and description definition (<dd>) tags. Think of the <dl> as a sort of key-value pair (grouping element), the <dt> as the column header, and the <dd> as the cell data.

Data grids are arranged into a maximum of 4 columns and are customizable by screen size. In the example below, the grid arranges itself into 1 column for extra-small screens, then splits into 2 columns for small screens, 3 columns on medium screens, and 4 columns for large screens.

Helpers

You can add all the standard helper classes to <dl>s inside a data grid. For example, if some condition is met such that a value should be a warning to the administrator, you can turn that definition list bright red.

Muted Text
Unimportant or disabled text
Success Text
Value that is good
Danger Text
Bad or problematic value
<div class="data-grid">
	<dl class="text-muted">
		<dt>Muted Text</dt>
		<dd>Unimportant or disabled text</dd>
	</dl>
	<dl class="text-success">
		<dt>Success Text</dt>
		<dd>Value that is good</dd>
	</dl>
	<dl class="text-danger">
		<dt>Danger Text</dt>
		<dd>Bad or problematic value</dd>
	</dl>
</div>

Site Header

Global Header

Use this .navbar construct at the top of the page to establish a global header for the admin interface. The club name should be wrapped in a .navbar-brand link, which links back to root (or the home page). The payments and logout buttons are contained in a .navbar-right component, which pins them to the right on larger screens.

This .navbar component also contains a .navbar-alert, which you can use at your discretion to post notices to the very top of the page at a global level.

Global alert, if you need it
Club Stabertson
  
<div class="navbar navbar-default">
	<div class="navbar-alert navbar-alert-muted">Global alert, if you need it</div>
	<div class="container-fluid">
		<a class="navbar-brand" href="/">Club Stabertson</a>
		<div class="navbar-right">
			<a class="btn btn-link navbar-btn" title="Payment Info"><span class="glyphicon glyphicon-usd"></span></a>
			<a class="btn btn-link navbar-btn" title="Log Out"><span class="glyphicon glyphicon-log-out"></span></a>&nbsp;
		</div>
	</div>
</div>
